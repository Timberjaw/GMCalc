Variables

Variables allow you to store numbers and equations for reuse. There are two system-defined variables available:

ans	Contains the resulting value from the last executed equation
last	Contains the original text of the last equation

Note that repeated use of last (i.e. by repeating the same equation over and over) may result in program lag, as the resultant equation will become longer and longer. To give an example:

2d4
-> now 'Last' contains '2d4'
2d(Last)
-> now 'Last' contains '2d(2d4)'
2d(Last)
-> now 'Last' contains '2d(2d(2d4))'
etc

Use responsibly.

You can define your own variables using the Variable Editor available from the Tools menu. Select a variable by left-clicking to edit. Insert a variable into your current equation or delete a variable from the list by right-clicking on its row and selecting the appropriate option.

Note: ans and last cannot be altered or deleted.
